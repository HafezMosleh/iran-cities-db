# iran-cities-db

Comprehensive and up-to-date JSON/SQL database of Iranian provinces and cities.

## Features
- 🚀 Fast and optimized
- 🔒 Secure by default
- 🛠 Easy to configure

## Installation
```bash
git clone https://github.com/HafezMosleh/iran-cities-db.git
cd iran-cities-db
```
