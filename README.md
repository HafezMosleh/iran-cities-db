## 🌟 Overview
The most accurate, clean, and developer-friendly database of Iranian provinces and their respective cities. Perfect for dropdown menus, location-based services, and shipping calculations in Iranian e-commerce platforms.

## ✨ Key Features
- 📑 **Highly Structured**: Formatted in clean, nested JSON.
- 🔄 **Up-to-Date**: Includes the latest provincial and municipal divisions.
- 🚀 **Plug & Play**: Ready to be imported into any NoSQL/SQL database.

## 🚀 Getting Started

### Prerequisites
Make sure you have `JSON_JavaScript` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/iran-cities-db.git
cd iran-cities-db
npm install
```

### Usage
```bash
node main.js
```
